Tower Breakers

P1和P2玩一个游戏，P1先手：

  1. 有$N$个塔，每个高$M$；
  2. 轮流操作，可以选择个高为$x$的塔，降低它的高度到$y$，需要$1\leq y \lt x$且$x%y==0$；
  3. 当前玩家不能移动时输

有几种情况：

  1. $M=1$时，因为不能找到小于$M$的数字，所以P2赢。
  2. $N % 2 = 0$，假设分成两堆分别有$N/2$个塔，不管P1怎么操作，P2都可以照着P1操作，所以P1必输。
  3. $N % 2 \neq 0$，P1随便选一个塔把它高度变为0，剩下偶数个塔，先手必输，所以P1必胜。

如果使用grundy number, 结果就是:

$$
\underbrace{g(m)\oplus g(m) \oplus \cdots \oplus g(m)}_{N个g(m)} =
\begin{cases}
g(m)\gt0, & N%2==1\ and\ m>1\
0, & otherwise
\end{cases}
$$

Introduction to Nim Game

  1. 两个玩家P1和P2轮流操作
  2. N堆石子，每堆的数量可能不同
  3. 轮流从某一堆中拾取1个或者多个
  4. 拿走最后一个石子的玩家胜

定义$nimsum$即所有数量的异或，$nimsum(X) = X_1\oplus X_2\oplus\cdots\oplus X_n$，

那么$nimsum=0$的局面是必败局面，$nimsum>0$的局面是必胜局面。

因为所有$nimsum>0$的局面都可以转化为$mnimsum=0$的局面，而$mnimsum=0$不管怎么操作都会变成$mnimsum>0$的局面。

  1. $mnimsum=0 \Rightarrow mnimsum>0$
    明显不管改变哪一个bit，都会使得$nimsum\neq0$

  2. $nimsum>0 \Rightarrow mnimsum=0$
    假设当前状态是$s=nimsum(x_1, x_2, \cdots,x_k, \cdots x_n)$，且$mnimsum>0$，从第$k$堆拾取之后之后变成$t=nimsum(x_1, x_2, \cdots, y_k, \cdots x_n)$。

    可以得到

    $$
    t = s\oplus x_k \oplus y_k
    $$

    因为$y_k=s\oplus x_k$时，$t=s\oplus x_k \oplus y_k=s\oplus x_k\oplus s\oplus x_k=0$。
    找到$s$最左的非0位，设为第$d$位，找到第$k$堆它的第$d$位也不是0；$k$一定存在，否则$s$的第$d$位必是0，所有0的异或值仍然是0；此时$s\oplus x_k < x_k$，因为$s$和$x_k$的前$d-1$位都是0，且$s$和$x_k$的第$d$位是1,$s\oplus x_k$的第$d$位变成了0。

和标准的NIM很像，除了每堆石子都只有一个的情况：如果有奇数堆，P1拿走最后一个输，否者胜。

其他情况计算nimsum即可，nimsum大于0时候P1胜。

Nimble Game

有$N$个方块，编号$0, 1, \ldots, N-1$，方块$i$上面有$C_i$个硬币。

每个回合从$i$移动一个硬币到$j$，$0 \leq i \lt j$。

所以硬币都移动到方块$0$时游戏结束，最后一个不能移动的选手输。

转化为标准的nim：

因为位置$i$可以移动到$[0, i-1]$的任意位置，相当于标准nim中一堆有$i$颗石子。

位置$i$有$C_i$颗硬币，相当于有$C_i$堆石子，每堆有$i$颗。

$C_i$个$i$的异或结果

$$
\underbrace{i\oplus i \oplus \cdots \oplus i}_{C_i\ of\ i} =
\begin{cases}
i, & C_i%2==1 \
0, & C_i%2==0
\end{cases}
$$

所以只需要考虑$C_i %2 = 0$的情况。

统计其中有多少个质数即可，如果有奇数个，先手必胜

Poker Nim

$N$堆石子，轮流操作，每个回合可以：

  1. 往第$i$堆添加$j \gt 0$颗石子
  2. 从第$i$堆移除$j \gt 0$颗石子
  3. 为了保证游戏结束，不能往任意一堆添加超过$k$次
  4. 拿走最后一颗石子胜利

和标准的NIM一样，因为如果当前玩家处于必胜态，如果对手添加石子，当前玩家把到手添加的石子移除即可。

Digits Square Board

在$N\times N$ 的板子中每一格写上一个数字，轮流操作：每次选择一块至少有一个数字不是质数的板子，把它按水平方向或者垂直方向随便切成两块。

直到没有板子可以切分时，游戏结束；第一个不能操作的选手输。

把质数换成0，合数换成1；计算所有的grundy number即可，复杂度$O(N^5)$

Fun Game

有两个数组$A, B$，$P1, P2$轮流操作：

  1. 选取一个没用过的$i$，每个$i$只能选一次
  2. 如果当前是$P1$，获得$A_i$分，否则获得$B_i$分，
  3. 最后分数高的胜利

当前玩家选择使得$A_i+B_i$最大的$i$，这样要么可以获得最大的分数，要么可以阻止对手获得大的分数。

Stone Division

一堆石子有$N$颗，一个集合$S={S_1, S_2, \cdots, S_m}$，轮流操作，选择一堆石子和$s=S_i$，把它成相等的$k$份，如果无法分割则输。

设$g(x)$是某一堆数量为$x$的石子grundy number，
$$
g(x)=\underbrace{g(\frac{x}{k}) \oplus g(\frac{x}{k}) \cdots \oplus g(\frac{x}{k})}_{d=\frac{x}{k}\ of\ g} \
=
\begin{cases}
g(d), k%2=1 \
0, otherwise
\end{cases}
$$

Chocolate in Box

Kitty and Katty

盒子里有$N$个数字，每次拿两个数字$A, B$出来，然后放回$A-B$，Kitty先手，最后只剩一个数字$X$：

  1. $X % 3 = 1$, Kitty胜利
  2. $X % 3 = 2$, Katty胜利
  3. $X % 3 = 0$, 最后一个移动的玩家胜利

考虑特殊情况$N=1$，都不能移动，所以$X % 3=1$是Kitty胜利。

$N\gt1$时候，最后总会剩下两个数字$A, B$，考虑他们除以3的余数$a, b$：

  1. 如果$a=b$，当前玩家胜利
  2. 如果$a\neq b$，当前玩家依然胜利，因为他可以用$a-b$或者$b-a$得到想要的数字，使得余数是$1$或者$2$。

所以如果$N$是偶数Kitty胜利，否则Katty胜利。

Katty

Powers Game

式子$2^12^2\cdots2^n$ ,P1,P2轮流把其中一个$*$变成$+$或者$-$，如果最终的结果能被17整除P2胜利，否则P1胜利。

结论很简单：如果$n%8=0$P1胜利，否则P2胜利。

Bob and Ben

$N$颗树，同上，每次一个结点以及以他为根的子树，最后一个不能移动的输。

计算每颗树的grundy number

  1. 树有$1$个结点， 只能变成$0$个结点, 所以Grundy number 是$1$.
  2. 树有$2$个结点，只能变成$1$个结点, 所以Grundy number 是$0$.
  3. 树有$X \gt 2$个结点， 可以变成$X’ \in {0, 1, 2, \cdots, X-1}$个结点，所以Grundy number是 $(X-1) % 2+1$.

Tower Breakers - The Final Battle

塔初始高度$H$，每一轮$P1$选择一个塔，把它分成$x$份，且$H=H_1+H_2+\cdots+H_x$，然后$P2$选择一个$P1$拆分出来的塔$H_y, 1\leq y \leq x$，$P1$需要支付$P2\ y^2$金钱，两人都使用最优的策略，$P1$尽量少付钱，$P2$ 尽量多收钱。最后$P2$能获取多少金钱？

$P2$不能获得超过$4 logH$的金币，因为如果每次$P1$都把塔分成$H-\frac{H}{2}$和$\frac{H}{2}$，最多$logH$次，每次获取$4$个金币。

设$DP_x$表示在$P2$最多获取$x$个金币的情况下塔的最高的高度。

$DP_0=1$

$DP_x=\sum_{i=1}^{\sqrt x}DP_{x-i^2}$
