Trailing Loves (or L’oeufs?)

题目描述

计算$N!$在$B$进制下末尾有几个0.

输入格式

两个整数$N$和$B$

输出格式

输出一个整数表示$N!$在$B$进制下末尾有几个0.

数据范围

  1. $1 \leq N \leq 10^{18}$
  2. $2 \leq B \leq 10^{12}$

样例数据

输入

输出

分析

设$S=N!=\sum{a_iB^i}$ 末尾有$K$个0,那么$S \mod B^K = 0$, 问题变成了求$K$的最大值.

可以对$B$进行质因数分解得到集合$Y$,然后计算$S / y, y \in Y$.

而$S/y = \sum_{i=0}^{log_yN}{S/y^i}$.
