CCIRCLES

题目

平面上有$N$个圆。我们称圆$i$和$j (i \neq j)$ 形成一个好的二元组，当且仅当可以在圆$i$选择一点$P_1$，圆$j$上选择一点$P_2$，使得$P_1$和$P_2$的欧氏距离恰好为$K$。注意$P_1$和$P_2$可以不是整点。
你需要回答$Q$个询问。每个询问给定目标距离$K$，你需要求出此时有多少好的二元组。请注意，可能有多个圆重合。

输入格式

输入的第一行包含两个整数$N$和$Q$，分别代表圆和询问的个数。
接下来$N$行，每行包含三个整数$X, Y, R$，代表一个圆心为$(X, Y)$，半径为$R$的圆。
接下来$Q$行，每行包含一个整数$K$，描述一个询问。

输出格式

对于每个询问，输出一行，包含一个整数，代表好的二元组数。

数据范围

$2 \leq N \leq 10^3$
$1 \leq Q \leq 5 \times 10^5$
$|X, Y| \leq 2 \times 10^5$
$1 \leq R \leq 2 \times 10^5$
$0 \leq K \leq 10^6$

输出样例

样例解释

圆1上的点(0, −5)和圆2上的点(8, 1)的欧氏距离为10。但是，这两个圆上任意一对点的距
离都不是0或者20。

分析

  1. 两个圆的位置有三种：包含、相交、相离。我们先考虑两个圆上两点的距离，距离是连续的，肯定是在最小距离和最大距离之间。假设两个圆分别是$(x_1, y_1, r_1)$和$(x_2, y_2, r_2)$，且$r_1 \leq r_2$，两个圆心距离是$D=\sqrt{(x_1-x_2)^2+(y_1-y_2)^2}$，下面分别讨论三种情况下两点距离的最小值$L_{min}$和$L_{max}$：
    • 包含：$L_{min}=r_2-r_1-D$，$L_{max}=D+r_1+r_2$
    • 相交: $L_{min}=0$，$L_{max}=D+r_1+r_2$
    • 相离：$L_{min}=D-r_1-r_2$，$L_{max}=D+r_1+r_2$
  2. 通过计算每两个圆之间的距离的最小值最大值，可以得到$O(N^2)$个二元组，可以当成成$O(N^2)$个线段区间处理，时间复杂度$O(N^2)$。问题转化成对每个要查询的值，有多少个线段包含它。
  3. 把要查询的点和线段区间的点放到一起排序，之后从小到大遍历，遇到一个线段的起始点计数加一，遇到结束点计数减一。时间复杂度$O((Q+N^2)log(Q+N^2))$，这里的距离是小数不会影响，因为查询是整数，整数和小数不会重合。
  4. 心得，遇到这种问题要仔细分析复杂度，如果有case跑不过不要慌。
