Number of Subarrays with Bounded Maximum

Posted on 2018-10-24

题目

给一个只包含正整数的数组，和两个正整数$L,R (L \leq R)$。
返回连续的不为空的子区间的个数，这些子区间中的数字的最大值不小于$L$且不大于$R$。

输入

输出

Note

  • $0 \leq L,R,A[i] \leq 10^9$
  • $A$的长度在在区间$[1, 50000]$

分析

数组长度大于1000，明显不能用$O(N^2)$的方法。
尝试DP，设$dp[i]$表示以$A[i]$结尾的合法的子区间的个数。
用$A=[2, 1, 4, 2, 3], L=2, R=3$做例子

  1. 如果$A[i] \lt L$
    例如$i=1$，只能把$A[i]$添加到以$A[i-1]$结尾的子数组中，所以$dp[i] = dp[i-1]$
  2. 如果$A[i] > R$
    例如$i=2$，以$A[i]$结尾的子数组都不满足要求，所以$dp[i] = 0$。同时记录$prev = i$。
  3. 如果 $L \leq A[i] \leq R$
    例如 $i=4$，这种情况下，任何从前一个不合法的位置之后的位置开始的以$i$结尾的子数组都满足要求，即$A[l\ldots i], (prev \lt l \leq i)$都满足要求，所以$dp[i] = i-prev$。
    最后$dp$的值相加就是结果，因为$dp[i]$只依赖于$dp[i-1]$和$prev$，所以可以把空间减少到$O(1)$。
