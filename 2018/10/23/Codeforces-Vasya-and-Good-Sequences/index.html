Codeforces Vasya and Good Sequences

Posted on 2018-10-23

1058E Vasya and Good Sequences

描述

Vasya有一个由$n$个数字组成的数组，$a_1,a_2,\ldots,a_n$，可以对数组执行如下操作：选择一个数字然后交换它二进制表示中的任意两个bit的位置，
例如可以把$6(\ldots00000000110_2)$转换成$3(\ldots00000000011_2), 12(\ldots000000001100_2), 1026(\ldots10000000010_2)$等等。
可以对任意多的数字执行任意多的这种操作。

如果对一个数组执行上面的操作之后，它所有的数字进行异或操作之后的结果是0，Vasya就认为这个数组是好的。

对于一个给定的数组$a_1,a_2,\ldots,a_n$, Vasya想知道有多少个区间$(l, r) 1 \leq l \leq r \leq n$使得$a_l, a_{l+1}, \ldots, a_r$是好的。

输入

第一行是一个数字$n(1 \leq n \leq 3 \times 10^5)$。
第二行包含$n$个数字$a_1, a_2, \ldots, a_n (1 \leq a_i \leq 10^{18})$

输出

输出一个数字，表示有多少个$(l, r)$ 使得 $1 \leq l \leq r \leq n$ 并且 $a_l, a_{l+1}, \ldots, a_r$是好的。

分析

因为我们可以交换任意bit，所以我们只需要知道每个数字的二进制表示中有多少个1就可以了。
先创建一个数组$b_1, b_2, \ldots, b_n$，其中$b_i$是$a_i$的二进制表示中1的个数，下面只考虑$b$。

先找到一种方法确定一个子段是否是好的，可以证明必须满足两个条件：这个子段的$b_i$的和必须是偶数；$b_i$的最大值必须小于等于其他元素的和。

按逆序遍历区间的左边界$l$，计算有多少个$r$使得$[l, r]$是好的。首先不管第二个条件，设$cntAll(l)$ 是使得$sum(b_l, b_{l+1}, \ldots, b_r)$的和是偶数$r$的个数。设$S_0, S_1$分别是后缀的和$\sum_{i=l}^n b_i)$是偶数、奇数的$l$个数。如果当前的和$\sum_{i=l}^n b_i$是偶数，那么$cntAll(l) = S_0$，否则$cntAll(l) = S_1$，因为$\sum_{i=l}^r b_i = \sum_{i=l}^n b_i - \sum_{i=l+1}^r b_i$。

在考虑第二个条件，把不满足第二个条件的子区间去掉。因为$a_i \leq 10^{18}$ 所以$b_i \lt 61$，所以如果子区间的长度 $\geq 61$的时候一定满足第二个条件。所以我们只需要遍历区间$[l, l+61]$减掉其中不满足条件的区间数量即可。

思考

输入规模$3 \times 10^5$，所以只能考虑$O(N)$的方法。
注意总结规律，最重要的是如何确定区间$[l, r]$是合法的。
