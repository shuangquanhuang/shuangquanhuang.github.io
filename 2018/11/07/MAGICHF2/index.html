MAGICHF2: 魔术师 vs 大厨 2

题目描述

上回，大厨在比特国打败了魔术师，并获得了一枚金币。
魔术师还不想这么快认输，所以他给了大厨 N 枚硬币，其中只有一枚是真正的金币，剩下的
N − 1 个都是赝品。赝品的重量都相同，且与真品不同。仅凭外观是分辨不出真假的。
大厨需要找出真品。公平起见，魔术师给了大厨一个天平。这是一个魔法天平，当两端重量
相同时天平保持平衡，但如果重量不同，天平会随机向一边倾斜。大厨最多用天平称重 K 次。要是 K 次称重之后大厨还不能锁定真品，他就得猜了（但大厨自己可以决定怎么猜）。
如果大厨以最优策略使用天平，他能选出正确硬币的最小概率最大是多少？我们想知道的是
最坏情况下的最好概率。

输入格式

输入的第一行包含一个整数 T，代表测试数据的组数。接下来是 T 组数据。
每组数据仅有一行，包含两个整数 N 和 K。

输出格式

对于每组数据，输出一行，包含一个整数，代表大厨找到金币的概率。只有当你的输出与标
准答案的绝对误差不超过 $10^{-6}$ 时，才会被判为正确。

数据范围

  1. $1 \leq T \leq 2 \times 10^5$
  2. $1 \leq N \leq 10^{18}$
  3. $0 \leq K \leq 10^9$

样例解释

  1. 第一组数据：大厨任选两个硬币放在天平的两侧。如果天平倾斜，则其中一个是真的（否则，剩下的一枚就是真的。但我们不考虑这一情况，因为它不是概率最小的）。大厨无法再使用天平了，因此只能猜。猜中的概率为 1/2。
  2. 第二组数据：与之前相同，大厨任选两个硬币放在天平的两侧。如果天平倾斜，那么大厨将其中一个硬币换成之前没称的硬币。如果天平再次倾斜，那么没有被替换的硬币是真品；否则被替换了的硬币是真品。

分析

每次操作之后剩下最少的硬币，一定是最优的。
有意思的是已经确定是假币的硬币也可以利用起来。

  1. 如果当前不确定的硬币数量$N$是奇数，那么可以用一半放到天平上，这样称一次之后就能确定在$(N+1)/2$个硬币中。
  2. 如果当前硬币数量$N$是偶数，分三种情况：
    2.1. $N$是4的倍数，分为 $N/4, N/4, N/2$三堆，称一次可以排除$N/2$个。
    2.2. $N$不是4的倍数，且没有已经找出的假币，会剩下$N/2+1$，比如当前有6个，只能分成$1, 1, 4$三堆，最坏情况剩下4个不能确定。
    2.3. $N$不是4的倍数，但是有有已经找出的假币，只会剩下$N/2$个，还是假设当前有6个，可以分成$1, 2, 3$三堆，再从假币中拿一个出来就变成了$2, 2, 3$，称一次之后就能排除一半。
